# 2026 区块链求职型学习计划总览

## 目标

- 在 2026 年 12 月前，达到可投递远程兼职区块链工程师的水平。
- 主线定位：Go 后端 + EVM 工具 / 数据 / 基础设施方向。
- 差异化方向：持续阅读 `go-ethereum`，做 1 个偏底层展示项目。

## 使用方式

- 每个月先看对应月份目录，明确本月目标。
- 每周进入对应周目录，按 `每日任务执行清单.md` 执行。
- 每天完成后勾选任务，并在文末补 3 行：
  - 今天完成了什么
  - 今天卡住了什么
  - 明天第一件事做什么

## 月度里程碑

- 2026-04：补 Go 与 Ethereum 地基，跑通本地 `geth`
- 2026-05：掌握 RPC、receipt、logs，完成链上查询 CLI
- 2026-06：完成事件监听器后端项目
- 2026-07：补工程化、文档、简历、GitHub 展示
- 2026-08：开始系统阅读 `go-ethereum`，做底层工具
- 2026-09：整理作品集，开始小规模投递
- 2026-10：争取开源参与、PR、bounty 或真实协作
- 2026-11：集中准备面试、试工与高频问题
- 2026-12：冲刺转化为兼职、contract 或 bounty 收入

## 每周固定节奏

- 周一到周五：学习 1 小时 + 编码 1 小时
- 周六：集中做项目 3 小时
- 周日：复盘、写周报、整理 README 和笔记

## 每周最低产出

- 1 次代码提交
- 1 次学习笔记或复盘
- 1 次对外可展示更新
- 1 次与真实机会相关的推进

## 求职检查项

- [ ] GitHub 主页持续更新
- [ ] 至少 3 个可展示项目持续推进
- [ ] 中英文简历持续迭代
- [ ] 有投递记录表
- [ ] 有英文自我介绍
- [ ] 有 1 个偏底层亮点项目
