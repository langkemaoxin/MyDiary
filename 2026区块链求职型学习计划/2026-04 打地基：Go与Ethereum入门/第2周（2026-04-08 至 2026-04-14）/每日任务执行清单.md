# 2026-04 第 2 周执行清单

## 一周目标

- 学会 struct、method、interface、pointer、module、test，能写一个小型 Go CLI。

## 必读资料

- [A Tour of Go: Methods and interfaces](https://go.dev/tour/methods/1)
- [Effective Go](https://go.dev/doc/effective_go.html)
- [Go Tutorial: Create a module](https://go.dev/doc/tutorial/create-module)

## 本周必须完成的小 Demo

- 场景：模拟一个最常见的数据管理类后端工具，让你把 struct、interface、slice、map、test 真正用起来。
- 功能要求：
  - 支持新增联系人、查看全部联系人、按姓名查询、删除联系人。
  - 每个联系人至少有姓名、手机号、备注 3 个字段。
  - 至少把核心数据逻辑拆成一个可测试的独立包。
- 完成效果：
  - 能通过命令行完成一次完整的“新增 -> 查询 -> 删除”流程。
  - 至少写 3 个测试，覆盖新增、查询、删除。
  - README 中写清楚运行命令和示例输入输出。
